Quickstart

This page follows fastai’s quickstart page by quickly showing a few learning tasks. More will be added here as they are added to the library.

FastAI.jl’s learning tasks all use the same basic steps and code:

  • create a data container

  • create a learning method

  • create learner

  • call a fit method

  • make predictions or view results

In this quick start, we’ll show these steps for a wide range of difference applications and datasets. As you’ll see, the code in each case is extremely similar, despite the very different models and data being used.

using FastAI, Metalhead
import CairoMakie; CairoMakie.activate!(type="png")

Classification

path = joinpath(datasetpath("oxford-iiit-pet"), "images")
iscat(p) = isuppercase(pathname(p)[1])
data = Datasets.loadfolderdata(path, filterfn=isimagefile, loadfn=(loadfile, iscat))
method = BlockMethod(
    (Image{2}(), Label([true, false])),
    (
        ProjectiveTransforms((128, 128)),
        ImagePreprocessing(),
        OneHot()
    )
)
learner = methodlearner(method, data, Models.xresnet18(), ToGPU(), Metrics(accuracy), batchsize=8)
fitonecycle!(learner, 5)

df = loadfile(joinpath(path, "train.csv"))

We swap out the target block Label for a LabelMulti, but the learning method definition is otherwise the same:

method = BlockMethod(
    (Image{2}(), LabelMulti(unique(Iterators.flatten(eachobs(data[2]))))),
    (
        ProjectiveTransforms((128, 128)),
        ImagePreprocessing(),
        OneHot(),
    )
)
BlockMethod(Image{2} -> LabelMulti{SubString{String}})
callbacks = [ToGPU(), Metrics(accuracy_thresh)]
learner = methodlearner(method, data, Models.xresnet18(), callbacks...; validdata=validdata)
Learner()
fitonecycle!(learner, 5, 0.001)

Segmentation

path = datasetpath("camvid_tiny")
data = (
    loadfolderdata(joinpath(path, "images"), filterfn=isimagefile, loadfn=loadfile),
    loadfolderdata(joinpath(path, "labels"), filterfn=isimagefile, loadfn=f -> loadmask(f, classes))
)
classes = readlines(open(joinpath(path, "codes.txt")))
method = BlockMethod(
    (Image{2}(), Mask{2}(classes)),
    (
        ProjectiveTransforms((128, 128)),
        ImagePreprocessing(),
        OneHot()
    )
)
backbone = Models.xresnet18()
learner = methodlearner(method, data, backbone, ToGPU())
fitonecycle!(learner, 8)
