FastAI.savemethodmodel

savemethodmodel(path, method, model[; force = false])

Save a trained model along with a method to path for later inference. Use loadmethodmodel for loading both back into a session. If path already exists, only write to it if force = true.

If model weights are on a GPU, they will be moved to the CPU before saving so they can be loaded in a non-GPU environment.

JLD2.jl is used for serialization.
