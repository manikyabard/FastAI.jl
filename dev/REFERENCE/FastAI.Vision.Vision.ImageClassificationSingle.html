FastAI.Vision.ImageClassificationSingle

ImageClassificationSingle(size, classes; kwargs...)
ImageClassificationSingle(blocks[, data]; kwargs...)

Learning method for single-label image classification. Images are resized to size and classified into one of classes.

Use FastAI.Vision.ImageClassificationMulti for the multi-class setting.

Keyword arguments

  • computestats = false: Whether to compute image statistics on dataset data or use default ImageNet stats.

  • aug_projections = Identity: augmentation to apply during ProjectiveTransforms (resizing and cropping)

  • aug_image = Identity: pixel-level augmentation to apply during ImagePreprocessing

  • C = RGB{N0f8}: Color type images are converted to before further processing. Use Gray{N0f8} for grayscale images.
