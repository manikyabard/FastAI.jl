FastAI.Datasets.ImageFolders

ImageFolders(; labelfn = parentname, split = false)

Recipe for loading a single-label image classification dataset stored in a hierarchical folder format. If split == true, split the data container on the name of the grandparent folder. The label defaults to the name of the parent folder but a custom function can be passed as labelfn.

julia> recipeblocks(ImageFolders)
Tuple{Image{2}, Label}
