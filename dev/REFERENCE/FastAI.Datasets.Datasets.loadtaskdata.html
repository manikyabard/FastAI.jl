FastAI.Datasets.loadtaskdata

loadtaskdata(dir, Method::Type{LearningMethod})

Load a task data container for a learning method type Method stored in dir in a canonical format.

loadtaskdata(dir, ImageSegmentation; [split = false])

Load a data container for ImageSegmentation with observations (input = image, target = mask).

If split is true, returns a tuple of the data containers split by the name of the grandparent folder.

loadtaskdata(dir, ImageClassification; [split = false])

Load a data container for ImageClassification with observations (input = image, target = class).

If split is true, returns a tuple of the data containers split by the name of the grandparent folder.

dir should contain the data in the following canonical format:

  • dir

    • split (e.g. “train”, “valid”…)

      • class (e.g. “cat”, “dog”…)

        • image32434.{jpg/png/…}

        • …

      • …

    • …
