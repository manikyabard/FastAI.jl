FastAI.encodedblock

encodedblock(encoding, block)
encodedblock(encoding, blocks)
encodedblock(encodings, blocks)

Return the block that is obtained by encoding block with encoding E. This needs to be constant for an instance of E, so it cannot depend on the sample or on randomness. The default is to return nothing, meaning the same block is returned and not changed. Encodings that return the same block but change the data (e.g. ProjectiveTransforms) should return block.
